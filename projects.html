Fylio — Document Management

Document filing SaaS modelled on Companies House-style organisational hierarchy. The core backend challenge was designing a storage layer that enforces tenant isolation in PostgreSQL without leaking references across organisation boundaries, while encrypting documents at rest using Fernet with a per-tenant key strategy. APScheduler handles background jobs — retention policies, scheduled exports — without blocking request threads. Stripe subscription billing is integrated with webhook-driven state transitions so plan changes are reliable even when the user closes the browser mid-checkout.

Seek — Search Indexing Platform

Enterprise search platform built to index millions of records from external data sources with sub-second query latency. The core engineering challenge is keeping the search index in sync with source systems — implemented a watermark-based incremental sync with distributed Redis locks to prevent concurrent sync jobs from corrupting the index. PostgreSQL partial indexes and trigram FTS accelerate queries; Redis-backed caching with graceful degradation handles load spikes. Soft-delete pattern preserves audit trails while keeping results clean.

Python Flask PostgreSQL Redis APScheduler PostgreSQL FTS

FileFlow — Cloud File System

Built to understand what zero-knowledge encryption actually means in practice. Files are encrypted client-side with AES-256 before hitting the API — the server stores ciphertext and handles S3 object management, but never sees plaintext. Real-time collaboration runs over SignalR; the live code sharing workspace uses Monaco editor with operational-transform-style conflict handling.
